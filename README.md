# DevShop
 
## Project Description
 
DevShop is a simple e-commerce website developed as part of the Git & GitHub Mini Project. The objective of this project is to practice Git version control concepts such as branching, commits, pull requests, merging, merge conflicts, and repository management.
 
---
 
## Technologies Used
 
- HTML
- CSS
- JavaScript
- Git
- GitHub
